# Análisis de medios de pago en supermercados argentinos
### ¿Está muriendo el efectivo? | 2017–2025

## Descripción
Proyecto de análisis de datos end-to-end sobre la evolución de los medios de pago y canales de venta en supermercados argentinos entre 2017 y 2025, utilizando datos oficiales del Ministerio de Economía de la Nación Argentina.

## Preguntas del análisis
- ¿Está disminuyendo el uso del efectivo como medio de pago?
- ¿Qué medio de pago domina y cómo evolucionó?
- ¿Cómo creció el canal online frente al salón físico?
- ¿Hay estacionalidad en los medios de pago?

## Herramientas utilizadas
- Microsoft Excel
- Power Query (ETL)
- Power Pivot (modelado de datos)
- DAX (medidas analíticas)

## Estructura del proyecto
- `proyecto-analisis-medios-de-pago-de-supermercados.xlsx` — Dashboard interactivo con modelo de datos
- `proyecto-analisis-medios-de-pago-de-supermercados.docx` — Documentación técnica del proyecto

## Fuente de datos
Subsecretaría de Programación Macroeconómica — [datos.gob.ar](https://datos.gob.ar/dataset/sspm-ventas-supermercados)
